SmartServiceHub is a web-based service booking application developed using ASP.NET Core MVC and MongoDB. The project allows users to register, log in, and book different services online easily. It provides a simple and user-friendly interface for managing bookings. Users can view available services and track their booking history. An admin panel is included to manage users, services, and bookings. The project follows the MVC architecture for clean and organized development. MongoDB is used as the backend database for storing application data. HTML, CSS, JavaScript, and Razor Views are used for frontend design. Secure authentication and password hashing are implemented for user security. This project is useful for online service management and booking systems.
