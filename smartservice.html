Smart Service Hub

https://smart-service-hub-rho.vercel.app

Smart Service Hub is a vehicle service booking web application developed using React.js. This project allows users to register, log in, add vehicle details, and book car or bike services online easily. Users can book services like oil change, car wash, engine repair, and more. The website also includes an Admin Panel where the admin can manage bookings by accepting, rejecting, or deleting requests. Email notifications are sent to the admin using EmailJS whenever a booking is made. The website is fully responsive and works properly on mobile and desktop devices. This project improved my skills in React.js, routing, responsive UI design, and front-end development.
