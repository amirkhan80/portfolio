Bag Store Android App



Watch Demo Video Download APK https://drive.google.com/file/d/1BTOfmt32_nQyHUBhqvCGwwRTv4RuOnBW/view?usp=drive_link

Bag Store App is an Android application developed using Android Studio and Kotlin with client certificate. Users can view different bags, manage quantities, add products to cart, and make payments using multiple payment options. The app includes login functionality, cart management, total price calculation, and responsive user interface design. This project helped improve my Android development and UI design skills.
