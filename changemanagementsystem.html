Change Management System Change Management System is a web application used to manage and track changes inside an organization or company. This system helps users handle change requests, approvals, updates, and task management in an organized way. Users can create requests, monitor progress, and manage different activities related to business or system changes. The project uses MongoDB database for storing user data, requests, and management records securely. It also includes an admin access panel where the administrator can manage the system and monitor activities.
